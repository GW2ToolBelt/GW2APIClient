Information about a map objective.

Parameters

id

the objective's ID

type

the objective's type

owner

the objective's owner (i.e. "Red", "Green", "Blue", or "Neutral")

lastFlipped

the ISO-8601 standard timestamp of when the objective was last flipped

claimedBy

the guild ID of the guild that currently has claimed this objective

claimedAt

the ISO-8601 standard timestamp of when the objective was claimed

pointsTick

the amount of points per tick given by the objective

pointsCapture

the amount of points awarded for capturing the objective

guildUpgrades

the IDs of the currently slotted guild upgrades

yaksDelivered

the total number of shipments delivered to the objective
